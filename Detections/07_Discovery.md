
# Discovery

## Overview

Discovery techniques are used by adversaries to collect information about systems, users, networks, and security controls before conducting additional attack stages.

Discovery activity often appears as legitimate administration, making high-quality detection engineering and contextual analysis essential.

This document contains the planned discovery detections maintained within this repository.
