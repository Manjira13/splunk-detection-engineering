
# Collection

## Overview

Collection techniques are used by adversaries to gather information of interest prior to exfiltration. Collected data may include documents, browser data, credentials, screenshots, or archived files.

Detecting collection activity helps identify attackers before sensitive information leaves the environment.

This document contains the planned collection detections maintained within this repository.
