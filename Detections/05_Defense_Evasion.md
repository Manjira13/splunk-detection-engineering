
# Defense Evasion

## Overview

Defense Evasion techniques are used by adversaries to avoid detection, disable security controls, and hinder incident response efforts. These techniques are commonly observed during post-exploitation and ransomware attacks.

Monitoring changes to security products, logging mechanisms, and operating system defenses provides valuable visibility into malicious activity.

This document contains the planned defense evasion detections maintained within this repository.
